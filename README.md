# Credit Card Customer Segmentation

This project applies unsupervised machine learning techniques to segment credit card customers based on their financial behavior.

## Project Objectives

* Perform data preprocessing and cleaning
* Handle missing values
* Apply feature scaling
* Use K-Means clustering for customer segmentation
* Determine the optimal number of clusters using:

  * Elbow Method
  * Silhouette Score
* Visualize customer segments using PCA

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

The dataset contains customer credit card information such as balance, purchases, cash advance, and payment behavior.

## Results

The model successfully grouped customers into different clusters based on spending and financial patterns, helping identify customer behavior segments.
